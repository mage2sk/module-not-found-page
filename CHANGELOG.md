# Changelog

All notable changes to this extension are documented here. The format
is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] — Initial release

### Added
- **Custom 404 page** — replaces the default Magento CMS noroute page
  with a modern, centered design featuring a gradient 404 code,
  configurable heading, and subheading.
- **Integrated search bar** — visitors can search products directly
  from the 404 page without navigating away.
- **Dynamic category links** — automatically fetches up to 6 top-level
  store categories (active + included in menu) so visitors can browse
  popular sections.
- **Admin configuration** under
  `Stores > Configuration > Panth Extensions > 404 Not Found Page`
  with toggle controls for search bar, popular links, contact info,
  and customizable heading/subheading text.
- **Contact information section** — optional email display for visitor
  support, configurable per store view.
- **Fully responsive design** — optimized for desktop and mobile with
  icon-only buttons on small screens.
- **Self-contained CSS** — no external stylesheets or JavaScript
  dependencies; works on both Hyva and Luma themes.

### Compatibility
- Magento Open Source / Commerce / Cloud 2.4.4 - 2.4.8
- PHP 8.1, 8.2, 8.3, 8.4
